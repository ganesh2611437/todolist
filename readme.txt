TO-DO LIST APPLICATION

This application is a command-line based To-Do List that allows users to manage their daily tasks effectively.

FEATURES:
- Add a new task with a priority and an optional due date.
- View all tasks, along with their priority, due date, and completion status.
- Update the details of an existing task.
- Mark a task as completed.
- View tasks that are overdue based on their due date.
- Search for tasks by keywords.
- Save tasks to a JSON file to retain data between sessions.

GETTING STARTED:
1. Install Python 3 on your system from https://www.python.org.
2. Clone or download the source code to your local machine.
3. Open your command line interface (CLI) and navigate to the directory containing the source code.
4. Run the application by typing `python todo.py` (replace `todo.py` with the path to your Python script if necessary).

USING THE APPLICATION:
- Upon starting the application, you will be presented with a menu of options.
- Choose the appropriate number to add, delete, view, or update tasks.
- Follow the on-screen prompts to perform the desired operations.
- Use the 'Save & Exit' option to save your tasks to a file before exiting.

PREREQUISITES:
- Python 3.x
